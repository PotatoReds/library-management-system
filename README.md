# Library Management System

## Description
A Python-based library system using OOP.

## Features
- Add/remove/update books
- Add/remove/update members
- Issue and return books
- Error handling
- Unit tests

## OOP Concepts
- Classes & Objects
- Inheritance (if added)
- Polymorphism (display_info)

## Run
python3 main.py

## Tests
python3 -m unittest tests/test_library.py
